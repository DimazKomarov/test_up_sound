# Telegram-бот для треков Яндекс.Музыки

Бот принимает ссылку на трек Яндекс.Музыки и возвращает:
- название,
- артиста,
- длительность.

## Почему aiogram и yandex-music

- `aiogram` — современный и очень популярный фреймворк под Telegram Bot API с хорошей производительностью и удобной архитектурой.
- `yandex-music` — активная неофициальная библиотека, которую обычно используют для интеграции с Яндекс.Музыкой, так как официального публичного API для сторонних разработчиков нет.

## Запуск

1. Установите зависимости:
   ```bash
   uv sync
   ```
   или
   ```bash
   pip install -r requirements.txt
   ```
2. Создайте `.env` из шаблона:
   ```bash
   cp .env.example .env
   ```
3. Заполните в `.env` значения:
   - `BOT_TOKEN` — токен Telegram-бота от [@BotFather](https://t.me/BotFather)
   - `YANDEX_MUSIC_TOKEN` — OAuth-токен Яндекс.Музыки

4. Запустите:
   ```bash
   uv run python bot.py
   ```
   или
   ```bash
   python bot.py
   ```

## Возможные ошибки

- Важно: бот загружает `.env` с `override=True`, поэтому новые значения из файла будут применяться даже если переменные уже были заданы в системе ранее.

- `TelegramUnauthorizedError: Unauthorized` или сообщение `Неверный BOT_TOKEN...`:
  - проверьте, что `BOT_TOKEN` в `.env` актуальный и без лишних пробелов;
  - если сомневаетесь, перевыпустите токен через `@BotFather` и обновите `.env`.

- Ошибка `451 Unavailable For Legal Reasons` от Яндекс.Музыки:
  - это не ошибка ссылки, а региональное/правовое ограничение со стороны сервиса;
  - попробуйте другой аккаунт Яндекс.Музыки или другой IP/регион.
